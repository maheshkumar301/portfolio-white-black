# Maheshkumar S - Premium Creative Portfolio

This is an ultra-premium, highly interactive portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. 
It features a stark, high-fashion editorial theme characterized by massive typography, dynamic negative space, and Awwwards-style cursor interactions.

## Features & Interactions
- **Cinematic Text Reveals**: Blurred, staggered typography load-ins
- **Global Scroll Tracking**: Dynamic scroll progress indicator
- **Interactive Spotlight**: Click-to-expand accordion skills index for mobile and hover panel for desktop
- **Cursor Reveal**: Custom physics-based cursor image reveals for project links
- **Sticky Parallax Timeline**: Dual-column sticky scrolling for experience history

## Technologies Used
- **React**: Component-based UI rendering
- **Vite**: Lightning-fast build tool and development server
- **Tailwind CSS**: Utility-first styling with custom design tokens
- **Framer Motion**: Advanced scroll animations, spring physics, and gesture tracking

## Local Development
To run this project on your local machine:

1. Ensure [Node.js](https://nodejs.org/) is installed.
2. Open a terminal in this project directory.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## GitHub Pages Deployment
This project is configured to be seamlessly deployed to GitHub pages at `https://maheshkumar301.github.io/portfolio-white-black`.

To publish a new version of the site:

1. Ensure all your changes are saved.
2. Run the deploy command:
   ```bash
   npm run deploy
   ```
3. This command will automatically:
   - Build the production-ready assets (`npm run build`)
   - Create or update the `gh-pages` branch in your repository
   - Push the built assets to GitHub

*Note: Make sure this project is initialized as a Git repository and pushed to your GitHub remote (`origin`) before deploying.*
